# AI Fitness Coach – AGENTS.md

## 🧠 Project Overview
This project is an AI-powered fitness coach that helps users create workout plans, track progress, and receive personalized recommendations.

## 🎯 Goals
- Generate personalized workout plans
- Suggest exercises based on user goals
- Track user progress over time
- Provide simple and clear guidance

## 👥 Users
- Beginners
- Intermediate users
- People looking for guided fitness plans

## ⚙️ Main Features
- User authentication (login/register)
- Profile setup (age, weight, goals)
- AI-generated workout plans
- Exercise recommendations
- Progress tracking

## 🧩 Tech Stack
- Frontend: Next.js / React
- Backend: API routes / Node.js
- Database: Supabase (or similar)
- Auth: Supabase Auth

## 🤖 AI Behavior Guidelines
- Keep responses simple and easy to understand
- Avoid technical fitness jargon
- Provide step-by-step workout instructions
- Adapt recommendations based on user input
- Do not give medical advice

## 📊 Data Handling
- Store user profiles securely
- Track workouts and progress
- Do not expose sensitive user data

## 🔒 Security
- Protect user authentication
- Validate all inputs
- Use environment variables for secrets

## 🧪 Testing
- Test authentication flows
- Test workout generation logic
- Test API endpoints

## 📌 Notes
- Keep UI simple and user-friendly
- Focus on personalization
- Ensure fast response times
